The [Pascal VOC challenge](http://host.robots.ox.ac.uk/pascal/VOC/index.html) is a very popular dataset for building and evaluating algorithms for image classification, object detection, and segmentation. However, the website goes down like all the time. In case you need the file, here they are:

## VOC 2012 ##

- [Train/Validation Data (1.9 GB)](https://data.pjreddie.com/files/VOCtrainval_11-May-2012.tar)
- [Test Data (1.8 GB)](https://data.pjreddie.com/files/VOC2012test.tar)
- [Development Kit](https://data.pjreddie.com/files/VOCdevkit_18-May-2011.tar)
- [PDF Documentation](https://data.pjreddie.com/files/VOC2012_doc.pdf)

## VOC 2007 ##

- [Train/Validation Data (439 MB)](https://data.pjreddie.com/files/VOCtrainval_06-Nov-2007.tar)
- [Test Data With Annotations (431 MB)](https://data.pjreddie.com/files/VOCtest_06-Nov-2007.tar)
- [Development Kit](https://data.pjreddie.com/files/VOCdevkit_08-Jun-2007.tar)
- [PDF Documentation](https://data.pjreddie.com/files/VOC2007_doc.pdf)
