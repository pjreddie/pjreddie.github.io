The [COCO dataset](http://mscoco.org/) is an excellent object detection dataset with 80 classes, 80,000 training images and 40,000 validation images. This is a mirror of that dataset because sometimes downloading from their website is slow.

## Images ##

- [2014 Training images [80K/13GB]](https://data.pjreddie.com/files/train2014.zip)
- [2014 Val. images [40K/6.2GB]](https://data.pjreddie.com/files/val2014.zip)

## Annotations ##

- [2014 Train/Val object instances [158MB]](https://data.pjreddie.com/files/instances_train-val2014.zip)
- [2014 Train/Val person keypoints [70MB]](https://data.pjreddie.com/files/person_keypoints_trainval2014.zip)
- [2014 Train/Val image captions [18.8MB]](https://data.pjreddie.com/files/captions_train-val2014.zip)
- [2014 Testing Image info [0.74MB]](https://data.pjreddie.com/files/image_info_test2014.zip)
- [2015 Testing Image info [1.83MB]](https://data.pjreddie.com/files/image_info_test2015.zip)
